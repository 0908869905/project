## 功能特色

### 1. 輸入區域
- **清晰的標籤**: "輸入中文文字"
- **佔位符提示**: 例如：我喜歡學習手語
- **多行文字框**: 支援較長的句子輸入
- **即時驗證**: 確保輸入不為空

### 2. 生成按鈕
- **視覺突出**: 藍色背景，全寬度
- **互動效果**: Hover 時變深藍色，輕微放大
- **載入狀態**: 顯示 "生成中..." 提示
- **禁用狀態**: 輸入為空或正在處理時禁用

### 3. 結果展示

#### 原始文字
- 灰色背景框
- 顯示使用者輸入的原始中文

#### TSL Gloss
- 淺藍色背景框
- 等寬字體顯示
- 展示台灣手語標記法

#### 手語圖片
- 方形容器（aspect-ratio: 1:1）
- 最大寬度 2xl
- 圓角邊框和陰影
- 響應式設計

### 4. 回饋按鈕
- **👍 很好**: 綠色主題，表示滿意
- **👎 不好**: 紅色主題，表示不滿意
- **⚠️ 回報問題**: 黃色主題，回報錯誤
- **回饋後**: 顯示感謝訊息，按鈕禁用

## 技術實作

### Tailwind CSS 類別使用
```css
/* 主容器 */
.min-h-screen: 最小高度 100vh
.bg-gradient-to-br: 對角漸層背景
.from-blue-50.to-indigo-100: 藍到靛色漸層

/* 卡片 */
.bg-white: 白色背景
.rounded-lg: 大圓角

/* 按鈕 */

```

## 互動流程

1. **使用者輸入文字**
   ```
   使用者 → 輸入框 → 文字驗證 → 按鈕啟用
   ```

2. **點擊生成按鈕**
   ```
   按鈕點擊 → 按鈕禁用 → 顯示載入動畫
   ```

3. **API 請求處理**
   ```
   前端 → POST /api/translate-complete → 後端
   後端 → OpenAI GPT-4 (翻譯 Gloss)
   後端 → OpenAI DALL-E 3 (生成圖片)
   後端 → 回傳結果 → 前端
   ```

4. **顯示結果**
   ```
   隱藏載入動畫 → 顯示結果卡片 → 啟用回饋按鈕
   ```

5. **使用者回饋**
   ```
   點擊回饋按鈕 → POST /api/postprocess → 顯示感謝訊息
   ```
