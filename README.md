# 手語圖片生成器 (Sign Language Image Generator)

一個網站輸入文字就能生成一張手語圖片

## 功能特色

- 🤟 輸入任意文字，自動生成對應的手語圖片
- 🎨 每個字符都有獨特的視覺化手語表示
- 📥 支援下載生成的手語圖片
- 📱 響應式設計，支援手機和電腦
- 🌏 支援中文、英文及各種字符

## 使用方法

1. 開啟 `index.html` 檔案在瀏覽器中
2. 在文字輸入框中輸入想要轉換的文字
3. 點擊「生成手語圖片」按鈕
4. 查看生成的手語圖片
5. 點擊「下載圖片」按鈕即可儲存圖片

## 檔案結構

- `index.html` - 主頁面結構
- `styles.css` - 樣式設計
- `script.js` - 手語圖片生成邏輯

## 技術實現

使用 HTML5 Canvas API 為每個字符生成獨特的手語視覺表示，基於字符編碼創建不同的手勢圖案。

## 線上預覽

直接在瀏覽器中開啟 `index.html` 即可使用。
